Projetos Desenvolvidos


Assina IFSULDEMINAS

Descrição do projeto:

O projeto Assina IFSULDEMINAS foi desenvolvido com o intuito de criar um site simples, elegante e que atenda às necessidades da instituição. Desenvolvido pela Diretoria de Comunicação (DICOM) do Instituto Federal do Sul de Minas Gerais, o site visa oferecer a todos os colaboradores uma assinatura de e-mail bonita, limpa e de fácil identificação.

Principais tecnologias utilizadas:

HTML, CSS, JavaScript, Google Apps Script e Google Sites.

Acesso ao site:

O site somente está disponível apenas para servidores do IFSULDEMINAS. Porém, todo o código está disponibilizado no meu git hub: clique aqui!

Principais funcionalidades:

- Criar uma assinatura;
- Gerar uma pré-visualização (preview) dessa assinatura;
- Incorporar a assinatura ao e-mail institucional automaticamente.

Principais Dificuldades:

Sou ainda um iniciante e esse foi o meu segundo grande projeto solo. Estava começando os estudos de JavaScript, pelo que foi necessário utilizar IA para fazer essa parte do projeto de forma certa e eficiente, sendo essa a principal dificuldade. Também foi um ensinamento grande, pois foi o meu primeiro contato com o google apps script e google sites. Em questão das outras linguagens, foram mais tranquilas pois eu já estava acostumado a elas. Outra dificuldade que tive foi criar um layout a partir de elementos de texto e imagens, pois o Gmail não tem uma ferramenta eficiente para isso!

Kanban DICOM

Descrição do projeto:

Kanban DICOM é um projeto desenvolvido por e para a Diretoria de Comunicação do Instituto Federal do Sul de Minas Gerais. Ele foi desenvolvido e pensado para ser usado como um gerenciador de projetos do setor, funcionando como um quadro Kanban interativo.

Principais tecnologias utilizadas:

HTML, CSS, JavaScript, Google firebase e GitHub pages.

Acesso ao site:

O site somente está disponível apenas para servidores da DICOM. Infelizmente, não posso compartilhar o código pois há chaves sensíveis presentes nele. Também não posso dar acesso ao site pois ele tem todos os projetos da DICOM.

Principais funcionalidades:

- Criação de tarefas, acompanhamentos de tarefas em execução e controle de tarefas concluídas;
- Geração de relatórios;
- Sistema de Usuários.

Principais Dificuldades:

Esse foi o meu terceiro grande projeto aqui na DICOM! Uma das grandes dificuldades minhas é o JavaScript. Já comecei a estudar mas ainda preciso de muito auxílio da IA. Aprender a mexer no google firebase também foi uma dificuldade para mim, visto que eu nunca tinha tido contato com essa plataforma. Também houve alguma pedra no caminho ao configurar o Git hub Pages, mas nada de mais.
